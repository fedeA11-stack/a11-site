  `fontWeight: 450`), `#282328` hardcoded ~40×; unused `--font-system` theme
  var in `globals.css`.
- **Non-functional / inaccessible CTAs** (correctness/a11y, S): home CTA
  `<button>`s have no handler/href; studio "Read our Manifesto" is a `<div>`
  (not keyboard-accessible). Partly product-incomplete (placeholders).
